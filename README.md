# ReactAppLoader

**ReactAppLoader** is a reusable loading wrapper built with **React and TypeScript**.  
It displays a fullscreen loader with a logo before rendering the main application content.

The component is designed to improve user experience during initial app load, API requests, or any async initialization phase.
